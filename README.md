# (C#코딩) 에코 메신저

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 시용자 키보드 입력을 받아서 처리하는 프로그램
- 사용한 플랫폼:
 - C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
 - Label, TextBox, Button, ListBox
- 사용한 기술과 구현한 기능:
 - Visual Studio를 사용하여 Windows Forms 애플리케이션 개발
